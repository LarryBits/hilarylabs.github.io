Project 1: Jellyfin Server Setup

Jellyfin is a free and open source media system that allows to organize, stream and share personal media libraries.

It acts as a central hub for your movies, shows, music, and other media, accessible from various devices like computers, smartphones, smart TVs, and more. Jellyfin uses a client-server architecture, where the server software manages the media library and the client applications connect to it for playback.
